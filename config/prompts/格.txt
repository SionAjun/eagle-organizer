### 格- 候选缩窄规则

R1:  含 题-赛博朋克 → [格-赛博朋克2077, 格-命运2, 格-控制]
R2:  含 题-都市幻想 或 题-近未来 → [格-绝区零, 格-控制, 格-VA-11 HALL-A]
R3:  含 题-剑与魔法 + (角-立绘 或 类-角色原画) → [格-原神, 格-塞尔达王国之泪, 格-魂系]
R4:  含 题-科幻 + (物-机甲 或 类-机械设定) → [格-装甲核心6, 格-质量效应, 格-异形:隔离, 格-命运2]
R5:  含 题-末世 / 题-废土 / 题-后启示录 → [格-辐射系列, 格-逃离塔科夫, 格-死亡搁浅]
R6:  含 题-恐怖 或 氛-诡异 → [格-异形:隔离, 格-控制, 格-雨世界]
R7:  含 风-像素 或 风-2D横版 → [格-Hades, 格-雨世界, 格-VA-11 HALL-A]
R8:  含 题-奇幻 且不含 R3 条件 → [格-原神, 格-塞尔达王国之泪, 格-魂系, 格-Hades]
R9:  含 题-复古未来 → [格-辐射系列, 格-极乐迪斯科, 格-VA-11 HALL-A]
R10: 含 风-日系 → [格-明日方舟, 格-原神, 格-绝区零]

S1:  已含任意 格- → skip 格-（不再补）
S2:  含 类-实景参考 → skip 格-
S3:  含 类-照片 / 类-摄影 → skip 格-
S4:  含 类-游戏截图 → skip 格-

未命中任何规则 → 自由从全部 格- 中选，拿不准则不补。
