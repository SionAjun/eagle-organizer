# 维度组合说明

代-* 标签描述文化时空,可独立或叠加。例:
- 一个维多利亚建筑场景:代-维多利亚
- 一个穿和服的人物:代-日式
- 两者可共存:蒸汽朋克设定下的维多利亚日本场景 = 代-维多利亚 + 题-蒸汽朋克

服-* 只描述服装形制,不含文化时代。文化靠 代- 叠加:
- 一件维多利亚西装 = 服-西装 + 代-维多利亚
- 一件中式长袍 = 服-长袍 + 代-中式

姿-* 仅在有人物时使用。

职-* 是身份判断,不要细化到具体角色名。

# 重名标签的维度区分

以下标签后缀相同但前缀不同,各自语义独立,请按维度判断使用,可共存。

## 机甲(4 个维度可共存)
- 题-机甲:画面整体题材是机甲类(如机甲战争、机甲驾驶员叙事)
- 角-机甲:画面主体是机甲单位(机甲本身就是"角色")
- 物-机甲:画面里有机甲作为物体之一(背景里的机甲、零件展示)
- 服-机甲:人物穿着机甲外骨骼/动力装甲

例:一张机甲驾驶员从机甲中走出的图 → 题-机甲 + 角-机甲 + 服-机甲(三者共存)

## 战术装备(2 个维度可共存)
- 物-战术装备:画面主体是装备本身(桌面陈列、零件特写、武器库)
- 服-战术装备:装备穿戴在人物身上

例:一个穿战术背心的士兵旁边摆着备用装备 → 物-战术装备 + 服-战术装备

## 维多利亚(2 个维度可共存)
- 题-维多利亚:叙事类型属于维多利亚时代题材(如蒸汽朋克叙事)
- 代-维多利亚:画面元素具有维多利亚的视觉时空特征(建筑/服装/家具)

例:一个蒸汽朋克题材的维多利亚街景 → 题-维多利亚 + 代-维多利亚

## 太空(2 个维度可共存)
- 场-太空:画面是具体的太空场景(行星表面、空间站内部、太空舱内)
- 域-太空:画面所属的领域是太空(广义,包括科幻题材的"太空背景")

例:宇航员在空间站舱内的特写 → 场-太空 + 域-太空

## 战斗(2 个维度可共存)
- 氛-战斗:画面氛围是战斗中(紧张、动作感、烟火)
- 姿-战斗:人物的姿态是战斗动作(挥剑、瞄准、格斗)

例:一个战士在战火中持剑冲锋 → 氛-战斗 + 姿-战斗

---

## 通用原则
- 这些重名是有意保留的多维度标记,不是冗余
- LLM 应根据画面实际情况打多个维度的标签,不要二选一
- 不要因为前缀相似就跳过其中之一
