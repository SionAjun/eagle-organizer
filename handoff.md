### 给 Claude Code 的指令包

```
任务: 线 1 收尾验证 — prompts 外置兼容性 Jaccard 实测

前提: 当前 Claude Code 已切回 Anthropic Claude。本任务不调用 mimo API。

1. 准备
   5 个目标 item_id:
   MDLQW3127UVMN, MNIYMC2M2H0K9, MNHQPCPS0UGSM, MNHNYAIV6EH06, MNHNWSMHAPEBR
   在 batch_results_023.json(或实际所在 batch)查出每张旧 tag 集合,记为 OLD_TAGS

2. 用当前 config/prompts/ 重打这 5 张
   走完整流程:加载 system.txt + main.txt + 各 prefix 片段
   对每张图调一次 LLM,得到 NEW_TAGS
   不要写入 batch_results / progress.json
   输出临时文件 reports/jaccard_test_results.json

3. 计算 Jaccard
   逐张: J = |OLD ∩ NEW| / |OLD ∪ NEW|
   计算 5 张平均 Jaccard
   找出差异最大的图,列出 (NEW - OLD) 和 (OLD - NEW)

4. 输出报告 reports/jaccard_test.md
   表格: item_id | 旧标签数 | 新标签数 | 交集 | 并集 | Jaccard
   汇总: 平均 Jaccard / 是否达到 ≥0.9 / 最大差异图的 diff

5. 验收判断
   ≥0.9: 报告"线 1 验收通过",回对话
   <0.9: 不要自动修复,回对话写明:
     a) 平均值
     b) 哪张图掉得最厉害,差了哪些标签
     c) 初步怀疑(prompt 拆分时哪个片段可能漏了)

不要做的事:
- 不要修改 tags.json / rules.json / config/prompts/
- 不要重跑 batch / 不要 apply / 不要改 progress.json
- Jaccard <0.9 时不要自动修补
- 不要触发 --sync
```

### 后续分支

### config/prompts/派.txt 完整内容

```
派- 维度:美学流派/设计运动。可与所有其他前缀叠加,尤其鼓励与
类-UI / 类-排版 / 格- / 题- 同时打。

判定原则:能识别出某种"主义"或"运动"的视觉特征即打,无需画面
是该流派的纯粹代表。一张图可属多组,鼓励叠加。

【现代主义系】功能优先、几何理性、网格秩序
  派-瑞士国际主义  严格网格 + Helvetica/Akzidenz + 不对称留白
  派-包豪斯        几何基本形 + 原色 + 无衬线字体
  派-极简主义      大留白 + 单色 + 极少元素
  派-风格派        直线 + 三原色 + 直角分割(蒙德里安)

【后现代/反叛系】反规则、装饰回归、刻意粗糙
  派-Brutalism     粗野原始 + Default 字体 + 反精致
  派-解构主义      破碎 + 倾斜 + 文字解体
  派-后现代主义    引用混搭 + 反讽 + 无统一语言(子流派不明确时打这个)
  派-孟菲斯        俗艳撞色 + 几何图案 + 80 年代设计

【未来想象系】对"未来"的不同时代想象
  派-Atompunk         50-60s 原子时代,圆润流线 + 镀铬
  派-Dieselpunk       30-40s 机械工业,黄铜 + 蒸汽
  派-Solarpunk        生态乌托邦,绿植 + 暖光 + 木竹
  派-赛博朋克美学     霓虹紫蓝 + CRT/扫描线(视觉外观)
  派-Vaporwave        粉紫渐变 + 罗马柱 + 90s 数字怀旧
  派-赛博颓废         破败 + 绝望 + 反乌托邦(情绪基调)
  派-高科技低生活     精密科技与脏乱市井并置(场景特征)
  派-太空时代         60s 登月美学,银白 + 圆润舱体

【时代怀旧系】对特定年代的视觉复刻
  派-Y2K              2000 前后,泡泡感 + 金属反光 + 半透明
  派-Frutiger Aero    07-13 苹果/Vista,水滴 + 玻璃 + 蓝绿
  派-Lo-Fi            模拟磁带噪点 + 低保真 + 怀旧暖
  派-Art Deco         20-30s 装饰艺术,对称 + 金色 + 几何花纹
  派-世纪中期现代     50-60s 北欧家具/排版风

【实验/数字系】媒介性流派
  派-故障艺术  glitch、数据损坏、扫描线撕裂
  派-像素美学  有意识的 8-bit 限色 + 抖动美学
              (区别于 类-像素画 — 类是物料形式,派是美学倾向)

═══ 三胞胎叠加规则 ═══
派-赛博朋克美学 / 派-赛博颓废 / 派-高科技低生活 是同一美学家族的
不同侧重,鼓励叠加而非互斥。一张典型赛博朋克氛围图通常 2-3 个
同时打,不要强行只选一个。

═══ 与其他前缀的边界 ═══
派- vs 题-:派- 是视觉处理,题- 是内容/世界观。可叠加。
  例:角色设定图世界观为赛博朋克 → 题-赛博朋克;
      视觉是 Y2K 处理 → 派-Y2K
派- vs 风-:风- 限定为绘画技法(写实/卡渲/厚涂)。所有"主义"
  与"设计运动"走 派-,风- 不收。
派- vs 版-:版- 限定为版面结构(网格/留白/堆叠);流派属性走 派-。
  例:一张瑞士国际主义海报 → 版-网格 + 派-瑞士国际主义(都打)
派- vs 格-:可叠加,不冲突。
  例:《极乐迪斯科》UI 截图 → 格-极乐迪斯科 + 派-Brutalism
```
